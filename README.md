## To Run
1. Install Flask:
  ```
  pip install flask
  ```
2. Create a Virtual Environment (Optional but recommended):
  ```
  # On Windows
  python -m venv venv

  # On macOS/Linux
  python3 -m venv venv
  ```
3. Activate the virtual environment:
  ```
  # On Windows
  .\venv\Scripts\activate

  # On macOS/Linux
  source venv/bin/activate
  ```
4. Run the Flask App:
  ```
  flask --app shout run
  ```
